TraME-Rcpp

Watch Star Fork


TraME-Rcpp is an R interface to TraME, a templated C++ library for solving problems of equilibrium computation and estimation in consumer demand and matching frameworks via the Mass Transportation Approach.

Features

  • Seamless and efficient OO-based interaction using Rcpp modules.
  • Requires no use of (or knowledge of) C++ by the user.

License: TraME is licensed under the GNU General Public License (GPL) version 2, or (at your option) any later version.

Authors: Alfred Galichon and Keith O'Hara

Download and Installation


Package requirements: R (v3.40, or above) and Rcpp, RcppArmadillo, and digest.

  • Execute the following commands, post installation of R itself, and you should be good to go:
    • install.packages("RcppArmadillo")
install.packages("digest")

The source code is available on GitHub. You need to install TraME-Rcpp from source.

  • Perhaps the quickest way to install from source is via the 'devtools' package in R.
  • First, make sure you have the appropriate development environment; if you're using a Mac, get Xcode from the App Store; if you're using Windows, get Rtools.
    • Note: For Mac users, unless you've compiled R from source, you will need gfortran v4.8.2. This can be obtained using the following commands (from a terminal window):
      •  
curl -O http://r.research.att.com/libs/gfortran-4.8.2-darwin13.tar.bz2
sudo tar fvxz gfortran-4.8.2-darwin13.tar.bz2 -C /
  • Then simply run the following commands in R:
    • install.packages("devtools")
library(devtools)
install_github("TraME-Project/TraME-Rcpp")
